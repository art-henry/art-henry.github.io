Мене звати Генрік

Так, так, мене дійсно звати Генрік Мхітарян, і, якщо ви дочитали до цього місця, то, мабуть, я вас вже зацікавив. Я - веб-розробник із Одеси, і я вмію робити круті речі з HTML5, CSS (і так, я знаю SCSS і SASS), PHP (не вчився, але хто не ризикує, той не п'є шампанське), JavaScript та Git.
Я також вправно жонглюю інструментами типу VS Code, Kommo CRM, Figma, HeidiSQL та FileZilla. Вчився я на курсах "Сучасний JavaScript + Vue з нуля на реальних проектах" та "Фронтенд-розробка веб-сайтів", що робить мене майже ніндзя у світі фронтенду.
Готовий взятися за ваш проект прямо зараз. Я завжди в терміни і не зникаю, як Дэвід Коперфілд. Моя мета - не просто створювати сайти, а робити їх привабливими, швидкими, і такими, що викликають "вау"-ефект.
Я гарантую вам відповідальний підхід і увагу до деталей. Не беру передоплати, а замість цього готовий допомагати і консультувати з будь-якими завданнями.
Об'єднаємо наші зусилля, щоб зробити вашу ідею реальністю! А якщо ви дочитали до цього місця, то, мабуть, вам варто просто написати мені.

Веб-розробка
Ентузіазіст фронтенд-розробник з основами веб-технологій, таких як HTML, CSS та JavaScript. Маю вміння створювати візуально привабливі та інтерактивні веб-сайти для забезпечення високої задоволеності користувачів.
JavaScript
Володію JavaScript і зосереджуюсь на створенні ефективних веб-застосунків з мінімальним кодом. Моя ціль - працювати з чистим JavaScript, уникаючи використання плагінів для забезпечення високої продуктивності та простоти. Зацікавлений у участі у фронтенд-проектах, де можу внести свій вклад, використовуючи інноваційні підходи в цифровому світі.

HTML5

HTML служить основою для вашого веб-сайту або додатку, і з п'ятою версією я зможу створити високо оптимізовану для SEO структуру для вашого продукту.

CSS3

Ця мова стилізації дозволяє мені створювати будь-який вигляд для вашого веб-сайту або додатку. Єдиний обмежувач - ваша уява!

JavaScript

Ця мова програмування дозволяє давати життя практично будь-чому: слайдерам, вікнам, підказкам, вкладкам, отриманню даних з сервера та багато іншого.

PHP

Використовуючи PHP в проектах, я налаштував kommo інтеграцію, підключив форми зворотного зв'язку та працював з базами даних, не маючи формальної освіти з цієї мови.

Бази даних

Хоч я ще не професіонал у роботі з базами даних, я вмію підключатися до них і витягувати необхідну інформацію, наприклад, з таблиць. Цей навик дозволяє мені ефективно інтегрувати дані в веб-проекти, забезпечуючи потрібну функціональність та продуктивність.
